改良版：現在地へボタン + 自動追従 + オフライン背景（キャッシュ）
----------------------------------------------------------------
- [現在地へ] ボタンでいつでも画面を自車位置に戻せます。
- [自動追従] チェックONで走行中は常に現在地を中央へ（地図操作すると自動でOFF）。
- Service WorkerがOSMタイルとアプリ本体をキャッシュし、電波圏外でも既に表示した範囲は背景地図が出ます。

【初回セットアップ】
1) このフォルダで PowerShell:
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   .\get_deps.ps1   （※ 依存は index.html と同じファイル名のまま）

2) index.html を開く → ブラウザのメニューから「アプリをインストール」（PWA）で
   オフライン起動可能。

【完全オフライン地図について】
- 今版は“事前に表示した範囲”をキャッシュします。電波が届かない場所で初めて表示する範囲は背景が出ません。
- 完全オフライン用にPMTiles/MBTilesのローカル読み込みにも対応可能です。必要になったら声をかけてください。
